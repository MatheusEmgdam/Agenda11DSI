<h2>OBJETIVO</h2>
 Este projeto foi feito com o objetivo de compreender melhor o funcionamento das bibliotecas da linguagem python
  <h2>FUNCIONAMENTO</h2>  
Este projeto funciona com base no ciclo **Entrada → Processamento → Saída**

O programa foi feito para demonstrar o processamento e saída usando dicionários.
A entrada utilizada é demonstrativa para exibir os efeitos da biblioteca coloroma.
O processamento consiste nas seguintes etapas:
-Entrada de Sinal
-Verificação do nível do sinal
-Retorno do nível de sinal no terminal
